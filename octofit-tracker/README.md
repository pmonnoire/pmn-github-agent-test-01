# OctoFit Tracker

A fitness tracking app scaffold for the OctoFit Tracker project.

Purpose
- User authentication and profiles
- Activity logging and tracking
- Team creation and management
- Competitive leader board
- Personalized workout suggestions

Structure

```
octofit-tracker/
├── backend/
│   ├── venv/ (excluded)
│   ├── requirements.txt
└── frontend/
```
